MPForm-Dokumentation: Spezielle Einstellungen

Schwierigkeitsgrad B 

Startwert für Radio und Checkbox
Standardverhalten: Wenn dieses Feld leer bleibt, übergeben Optionsfelder und Kontrollkästchen den angezeigten Text.
Falls mit den Eintragungen aus dem Formular eine statistische Auswertung durchgeführt werden soll, können sind lange Sätze als Bezeichner der einzelnen Optionen ziemlich unhandlich. Einfacher ist es dann, wenn die übermittelten Ergebnisse einfach durchnummeriert sind. Das können Sie erreichen, indem Sie den Startwert für Radio und Checkbox auf den gewünschten Anfangswert setzen.
Falls hier ein Zeichen eingetragen wird (sinnvoll könnten z.B. 0, 1 oder a sein), wird dieser Wert für jede Option hochgezählt und dann statt des Textes zurückgegeben. Wenn Sie den Startwert auf 1 setzen und der Benutzer kreuzt die dritte Checkbox einer Frage an, so wird dafür eine 3 eingetragen. Würde der Startwert A lauten, so würde ein C in der Datenbank landen usw.
Value-Option Trenn-String
Wenn die Werte nicht durchnummeriert sein sollen, aber dennoch sich von dem beschreibenden Text unterscheiden sollen, können Sie diesen Trenn-String verwenden. Beispiel: Die Werte sind Preise mit Währung, (10 &Eurosign;, 15 &Eurosign;, 20 &Eurosign;) aber die Werte sollen ohne das Euro-Symbol in die Datenbank geschrieben werden. Dann geben Sie dem Optionsfeld den vollen String mit Währung und hängen den Value-Option Trenn-String daran an und dahinter noch einmal den Wert (an dieser Stelle ohne das Euro-Symbol). Welcher String als Trenner konkret verwendet werden soll, können Sie hier einstellen.
Datumsformat
Das Datumsformat für den Javascript-Kalender. Von Hand kann der Benutzer im Datumsfeld Freitext eingeben.
