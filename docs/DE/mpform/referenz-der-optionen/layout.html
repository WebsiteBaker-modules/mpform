 MPForm-Dokumentation: Layout

Schwierigkeitsgrad B

Das Formular wird so aufgebaut, dass vom Programm eine Schleife durchlaufen wird, und in jedem Durchlauf dieser Schleife wird ein Feld ausgegeben, indem die unter Feld Schleife angegebenen Platzhalter durch die entsprechenden Inhalte ersetzt werden. Enthält die Feldschleife den Platzhalter {TEMPLATE}, so erscheint beim Bearbeiten der Felder ein Eingabefeld, in dem Sie festlegen können, was für diesen Platzhalter eingesetzt werden soll. Verwenden Sie {TEMPLATE1}, {TEMPLATE2} usw., so bezieht sich dies auf die jeweilige Zeile aus diesem Eingabefeld.

In Kopfzeile steht der html-Code, der vor dem ersten Feld ausgegeben wird.

In Fußzeile steht der html-Code, der nach dem letzten Feld ausgegeben wird.

Standardmäßig wird ein div-basiertes Layout angeboten, man kann das Formular aber auch mit einer Tabelle aufbauen.
