MPForm-Dokumentation: Einstellungen für Datei-Upload

Schwierigkeitsgrad B

Falls es im Formular ein Feld des Typs Datei-Upload gibt, kann der Benutzer zusammen mit dem Formular eine oder auch mehrere Dateien auf den Server laden. Die Einstellungen zum Umgang mit dieser Datei legen Sie hier fest.

Die Bezeichner sollten selbsterklärend sein. Die Verzeichnis- und Dateiberechtigungen gelten nur für Server, die ein anderes Betriebssystem als Windows verwenden. Aus Sicherheitsgründen ist MPForm so programmiert, dass der Benutzer nicht erfährt, unter welchem Dateinamen und in welchem Verzeichnis die hochgeladene Datei auf dem Server gespeichert wurde.
