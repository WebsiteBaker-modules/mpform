MPForm-Dokumentation: SPAM-Schutz

Schwierigkeitsgrad B

Captcha-Prüfziffer
Sie gibt an, ob als zusätzlicher Schutz vor Spammern eine Antwort auf eine Prüffrage eingegeben werden muss. Dadurch soll verhindert werden, dass das Formular von einem Programm statt von einem Menschen ausgefüllt wird. Unter Admin-Tools - Captcha and Advanced-Spam-Protection (ASP) Control im Backend können Sie entsprechende Einstellungen auswählen.
Max. Eintragungen pro Stunde
Falls in einer Stunde mehr Einträge gemacht werden sollten als hier angegeben, werden diese nicht mehr akzeptiert. Man geht davon aus, dass ein plötzliches hohes Aufkommen von Forumulareintragungen auf einen Missbrauch zurückzuführen ist, bei dem das Formular automatisiert ausgefüllt wird.
Max. gespeicherte Eintragungen
Diese Option dient sowohl dem Spam-Schutz als auch dem Ziel, die Datenbank nicht ins unermessliche wachsen zu lassen. Den Wert müssen Sie an das erwartete Aufkommen anpassen, denn wenn er überschritten wird, werden einfach die ältesten Einträge in der Submissions-Tabelle gelöscht.
