MPForm-Dokumentation: Hilfetexte für Benutzer

Schwierigkeitsgrad B 

Hilfetexte, die den Benutzer beim Ausfüllen des Formulars unterstützen sollen, können im Backend für die einzelnen Felder hinterlegt werden. Innerhalb von dem Backend-Feld Feld Schleife müssen Sie den Platzhalter {HELP} platzieren. Dann wird an dieser Stelle für die Felder, die einen Hilfetext hinterlegt haben, ein Button mit einem Fragezeichen angezeigt. Falls der Benutzer JavaScript aktiviert hat und den Button betätigt, wird der Hilfetext angezeigt.

Darüber hinaus kann der Hilfetext auch direkt angezeigt werden, so dass der Benutzer nicht erst auf das Fragezeichen klicken muss. Verwenden Sie hierfür den Platzhalter {HELPTXT}. Wenn Sie diesen Platzhalter irgendwo in das Feld Feld Schleife im Backend platzieren und für einige Felder Hilfetext eingeben, werden alle diese Felder den Hilfetext sofort anzeigen.
