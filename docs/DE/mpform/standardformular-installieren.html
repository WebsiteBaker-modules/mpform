MPForm-Dokumentation: Standardformular installieren

  Schwierigkeitsgrad B 

Man kann ein Formular auch z.B. auf Ihrem lokalen Rechner vorbereiten und das fertige Formular auf dem Webserver importieren. Wenn Sie ein ganz gewöhnliches Kontaktformular installieren wollen, können Sie auch das Standardformular importieren:

  1. Laden Sie das Standardformular deutsch herunter.
  2. Legen Sie auf dem Zielrechner einen neuen Abschnitt vom Typ mpform an.
  3. Wählen Sie in diesem neu angelegten Abschnitt (oben in der Mitte) die zuvor heruntergeladene Datei mit dem Browser aus und klicken Sie anschließend auf Formular importieren.
  4. In den Optionen des Formulars müssen Sie nun noch Ihre email-Adresse eintragen.
