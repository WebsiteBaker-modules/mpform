MPForm-Dokumentation: Installation des Moduls

Schwierigkeitsgrad A 

Voraussetzungen für die Verwendung von MPForm

  • Website Baker 2.8.2 oder höher
  • PHP 5.1 oder höher

Erstmalige Installation

  1. Laden Sie die aktuelle Version auf Ihren PC herunter. Sie brauchen die zip-Datei nicht auszupacken.
  2. Melden Sie sich als Administrator am Backend von Website Baker an.
  3. Klicken Sie im Menü auf Erweiterungen und dann auf Module.
  4. Mit dem Button Durchsuchen wählen Sie die soeben heruntergeladene zip-Datei aus.
  5. Klicken Sie auf Installieren.
  6. Unter Details zum Modul wählen Sie nun MPForm aus und klicken auf Details. Wenn Sie hier etwas angezeigt bekommen, war die Installation bis dahin erfolgreich.
  7. Diese Dokumentation ist im Modul mit enthalten. Wenn Sie verhindern wollen, dass die Besucher Ihrer Webseite darauf Zugriff haben, können Sie das docs Unterverzeichnis mit Hilfe einer .htaccess Datei schützen.

Upgrade von einer älteren Version

Wenn Sie ein Upgrade durchführen, wissen Sie ja bereits, wie die Installation abläuft.

Das Upgrade führt keine Änderungen an folgenden Dateien durch:

  • frontend.css
  • backend.css
  • private.php

Sie könnten an diesen Dateien kundenspezifische Änderungen durchgeführt haben, die nicht verloren gehen sollen. Es ist Ihre Aufgabe, die vorhandenen Dateien mit denen vom Upgrade (die zusätzlich ".default." im Namen besitzen) zu vergleichen und zu entscheiden, ob die alte oder die neue Version verwendet wird oder ob beide Versionen zusaqmmengeführt werden müssen.
