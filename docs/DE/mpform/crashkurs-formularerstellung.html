MPForm-Dokumentation: So erstellen Sie Ihr erstes Formular

Schwierigkeitsgrad A

  1. Klicken Sie im Backend auf Seiten
  2. Geben Sie ganz unten unter Seite hinzufügen einen Titel ein, z.B. Testformular.
  3. Bei Art wählen Sie MPForm aus.
  4. Übergeordnete Datei und Sichtbarkeit legen Sie wie gewünscht fest.
  5. Klicken Sie auf Hinzufügen.
  6. Nun haben Sie ein leeres Formular, das mit Inhalt (Feldern) gefüllt werden muss. Klicken Sie dazu auf Feld.
  7. Bei Titel geben Sie Ihr Name ein und bei Typ: Textfeld (einzeilig)
  8. klicken Sie auf Speichern
  9. wählen Sie aus: Länge: 80  markieren Sie Pflichtfeld
  10. erstellen Sie ein neues Feld, indem Sie auf Feld kopieren klicken
  11. Ändern Sie bei dem neuen Feld den Titel in Ihre email-Adresse und den Typ in E-Mail-Adresse. Länge und Pflichtfeld lassen Sie stehen.
  12. kopieren Sie es erneut mit Feld kopieren
  13. Ändern Sie den Titel in Ihr Problem und den Typ in E-Mail-Betreff (Suffix). Länge und Pflichtfeld lassen Sie stehen.
  14. Selbstverständlich können Sie Felder auch Feld hinzufügen anklicken um neue Felder hinzuzufügen anstelle von bestehenden Feldern zu kopieren. Versuchen Sie dies und nehmen Sie dann die folgenden Einstellungen vor
  15. Titel: Schildern Sie Ihr Anliegen Typ: Textfeld (mehrzeilig)
  16. klicken Sie auf Speichern
  17. wählen Sie Pflichtfeld aus
  18. klicken Sie erneut auf Speichern
  19. schließen Sie mit Abbrechen den Vorgang ab.

Wenn Sie möchten, dass der Absender des Formulars eine Bestätigungsmail erhält, müssen Sie noch folgende Schritte durchführen:

  • Allgemeine Optionen
  • Scrollen Sie nach unten zum Abschnitt E-Mail-Einstellungen (Bestätigungsmail an Formularabsender)
  • Im Feld Empfängeradresse wählen Sie Feld: Ihre email-Adresse
  • Speichern

Nun haben Sie ein funktionsfähiges Formular mit 4 verschiedenen Feldtypen erstellt. Wenn Sie ganz oben im Backend auf das zweite Symbol von links klicken (falls Ihr Browser es unterstützt erscheint hier den Tooltip Ansicht), können Sie Ihr Formular bewundern und testen.

Natürlich kann man das Formular noch mit Hilfe diverser Einstellungen an die eigenen Vorstellungen anpassen. Auf weiteren Hilfeseiten sind die optionalen Einstellungen des Formulars im Detail beschrieben.
