MPForm-Dokumentation: Mehrseitige Formulare

Schwierigkeitsgrad A 

  1. Erzeugen Sie die einzelnen Seiten mit den Formularteilen
  2. Setzen Sie auf allen Seiten die Workflow-Einstellung Namensendung für die Ergebnistabelle auf den selben String.
  3. Auf der ersten Seite setzen Sie die Option Ist ein Folgeformular auf NEIN, auf allen folgenden Seiten aber auf JA.
  4. Auf allen Seiten setzten Sie die Folgeseite auf den nächsten Abschnitt des Fragebogens. (Auf der letzten Seite bietet es sich an, auf eine normale Seite weitzerzuleiten, auf der Sie dem Benutzer z.B. für seine Teilnahme an der Umfrage danken.)
  5. Nun können Sie die Formularfelder auf allen zuvor angelegten Seiten eingeben.

Bis mpform 1.2.x wurden in der Bestätigungsmail an den Formularabsender und in der Mail an den Seitenbetreiber nur die Felder der jeweils aktuellen Seite hinterlegt. Ab 1.3.x werden die Eingaben zwischen den verschiedenen Seiten eines mehrseitigen Formulars weitergereicht und die letzte Seite kann somit alle Eingaben gesammelt per Mail verschicken und in die Submissions-Tabelle eintragen.
