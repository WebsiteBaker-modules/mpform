MPForm Documentation: Settings of the module

There are various settings of MPForm. They have been divided into two pages, one with "standard" settings and the other one with advanced settings. The later contain those settings most users probably will never change. Another reason for the existence of two pages is to assist new users who probably will not need to change the advanced settings. They might get lost on a huge settings page which would also present the advanced settings in the first course.

The General Settings page contains the following sections:

  • Workflow Settings
  • Layout
  • E-Mail Settings (mail to site owner)
  • E-Mail Settings (confirmation mail to user)

The Advanced Settings page contains the following sections:

  • Field Formatting for Notifications
  • Settings for File Upload
  • SPAM Protection
  • Special Settings

In the CSS Settings you can comfortably edit the files frontend.css and backend.css.
