MPForm Documentation: Field Formatting for Notifications

Level of difficulty: B  

In MPForm it is possible to format the content of the presentation of a submission in emails, frontend and backend separately. The place holder {DATA} exists for the actual user input. It will be replaced by the contents of all fields. In the settings of this section it is configured how title and content of the different field types will be formatted. {TITLE} is the place holder for the title of the field (call it the question) and {DATA} represents the user input (the answer). {CLASSES} contains the appropriate classes for the respeitve field type.
