MPForm Documentation: Installation of the module

Level of difficulty: A

Preconditions for the usage of MPForm

  • Website Baker 2.8.2 or above
  • PHP 5.1 or above

First time installation

  1. Download the latest version. There is no need to unzip the file.
  2. Login as Administrator at the backend of Website Baker.
  3. Click at Add-ons in the main menu and then on Modules.
  4. With the button Search you choose the zip file you just downloaded.
  5. Click on Install.
  6. Choose MPForm under Module Details and click on View Details. If you can see some information the installation was successfull until this point.
  7. This documentation is included in the module. If you want to prevent it to be visible to the visitors of your web site you can protect the docs directory using a .htaccess file

Upgrade from an older version

If you do an upgrade, you already know how the installation works.

The upgrade will not change the following files:

  • frontend.css
  • backend.css
  • private.php

This is because you might have customized those files. It is your task to compare the files from the upgrade package (which additionally contain "default" in their names) and to decide whether to use the old or new versions or whether to merge both.
